**src/lib/evaluation/use-evaluation.ts** -- Custom React hook using useReducer (addresses review concern: useReducer maps 1:1 to SSE events instead of managing 8 separate useState calls).

```typescript
"use client";
import { useReducer, useCallback } from "react";
import type { EvaluationProgressEvent } from "./orchestrator";
import type { EvaluationOutput, EvaluationDimension, ProposalEvaluation } from "./schemas";
```

Define state type:
```typescript
type EvaluationState = {
  status: "idle" | "evaluating" | "evaluated" | "failed";
  completedDimensions: Map<EvaluationDimension, EvaluationOutput>;
  failedDimensions: Set<EvaluationDimension>;
  aggregate: { weightedScore: number; completedDimensions: number } | null;
  evaluation: ProposalEvaluation | null;
  ipfsCid: string | null;
  txHash: string | null;
  error: string | null;
  storageErrors: Array<{ step: "ipfs" | "chain"; error: string }>;
};
```

Define action type -- a discriminated union matching EvaluationProgressEvent plus a "reset" action:
```typescript
type EvaluationAction =
  | { type: "reset" }
  | EvaluationProgressEvent;
```

Define initial state:
```typescript
const initialState: EvaluationState = {
  status: "idle",
  completedDimensions: new Map(),
  failedDimensions: new Set(),
  aggregate: null,
  evaluation: null,
  ipfsCid: null,
  txHash: null,
  error: null,
  storageErrors: [],
};
```

Define reducer function `evaluationReducer(state: EvaluationState, action: EvaluationAction): EvaluationState`:
- `reset`: return `{ ...initialState, completedDimensions: new Map(), failedDimensions: new Set(), storageErrors: [] }` (fresh instances of mutable collections)
- `started`: return `{ ...initialState, status: "evaluating", completedDimensions: new Map(), failedDimensions: new Set(), storageErrors: [] }`
- `dimension_complete`: create new Map from state.completedDimensions, set the new entry `(action.dimension, action.output)`, return `{ ...state, completedDimensions: newMap }`
- `dimension_failed`: create new Set from state.failedDimensions, add `action.dimension`, return `{ ...state, failedDimensions: newSet }`
- `aggregate_computed`: return `{ ...state, aggregate: { weightedScore: action.weightedScore, completedDimensions: action.completedDimensions } }`
- `storage_failed`: return `{ ...state, storageErrors: [...state.storageErrors, { step: action.step, error: action.error }] }`
- `stored`: return `{ ...state, ipfsCid: action.ipfsCid, txHash: action.txHash }`
- `complete`: return `{ ...state, status: "evaluated", evaluation: action.evaluation }`
- `failed`: return `{ ...state, status: "failed", error: action.error }`
- default: return state (ignore unknown event types for forward compatibility)

The hook `useEvaluation()` returns:
```typescript
{
  startEvaluation: (proposalId: string, proposalText: string) => Promise<void>;
  ...state, // spread all EvaluationState fields
  isLoading: state.status === "evaluating",
}
```

`startEvaluation` implementation (wrapped in useCallback):
1. Dispatch `{ type: "reset" }` then dispatch `{ type: "started", proposalId, totalDimensions: 4 }`.
2. `const response = await fetch("/api/evaluate", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ proposalId, proposalText }) });`
3. If response not ok (400/429), parse error JSON, dispatch `{ type: "failed", error: data.error }`, return.
4. Get reader: `const reader = response.body?.getReader();` and `const decoder = new TextDecoder();`
5. SSE read loop with buffer for partial chunks:
   ```typescript
   let buffer = "";
   while (true) {
     const { done, value } = await reader.read();
     if (done) break;
     buffer += decoder.decode(value, { stream: true });
     const lines = buffer.split("\n\n");
     buffer = lines.pop() ?? ""; // keep incomplete chunk
     for (const line of lines) {
       const dataLine = line.trim();
       if (!dataLine.startsWith("data: ")) continue;
       try {
         const event = JSON.parse(dataLine.slice(6)) as EvaluationProgressEvent;
         dispatch(event);
       } catch {
         // Ignore malformed SSE events (T-02-10 mitigation)
       }
     }
   }
   ```
6. Wrap entire function in try/catch: on network error, dispatch `{ type: "failed", error: "Network error: connection lost" }`.
