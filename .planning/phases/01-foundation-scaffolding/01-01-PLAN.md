| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-01-02 | Tampering | package.json dependencies | mitigate | Pin major versions in package.json. bun.lock committed for reproducible installs (NOT in .gitignore). |
| T-01-03 | Information Disclosure | Solana keypair | mitigate | solana-keygen output goes to default ~/.config/solana/id.json (outside repo). .gitignore pattern *.json blocks accidental keypair commits. |
